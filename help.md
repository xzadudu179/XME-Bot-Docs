## 注意事项

### 提醒

- 请 **一定不要** 以及 **尽量避免** 让 bot 以任何形式输出敏感/违规字符，很容易导致 bot 被永久冻结！

  如果有任何形式导致 bot 因为用户输入的违规行为被封禁，一旦发现会 **追究责任** 且 **记录用户 qq 账号。**

- 该 bot 还在 **早期开发阶段**，很多功能很可能会变化很大。

### 关于指令

> 关于指令详情参见[指令术语](glossary?id=%e6%8c%87%e4%bb%a4)

- XME-Bot 的指令默认以 `/`、`!`、 `！`、 `.` 中任意一个的字符开头，在开头字符前 @ 机器人或输入类似 `xme,` 之类可被解析为呼叫机器人的语句时，会被判定为 `to_me` 也就是**呼叫机器人**。
- 在帮助的指令参数相关里，`<>` 括起来的参数代表[可选参数](glossary?id=可选参数)， `()` 括起来的参数代表[必要参数](glossary?id=必要参数)， `[]` 括起来的参数代表[类 Shell 参数](glossary?id=类-shell-参数)。

#### 指令格式

- 在指令列表中，指令格式为：

  **指令名**

  - **作用**

    关于指令是干什么用的。

  - **用法**

    关于指令的具体使用语法，例如：
    ``` Text
    /指令名 (必选参数) <可选参数> [类Shell参数]
    ```

  - **权限/可用范围**

    关于指令所需的[权限](glossary?id=权限)。若没有则为 `无`。若有会写出具体条件，需要符合条件才可调用指令。

- **别名**

    关于指令的别名，使用指令的别名也可以调用指令。

    比如 `help` 指令有 `帮助` 别名，所以  使用 `/帮助` 和使用 `/help` 并没有区别。

## 指令列表

---

### cancanneedwife

- **作用**

  查看指定群员今日的老婆群员。

- **用法**

``` Text
/cancanneedwife (@一个群友)
```

- **权限/可用范围**

  - **在群聊内**

- **别名**

  `看看老婆`、 `peekwife`、 `kkndwife`、 `kkndlp`、 `kklp`。

---

### dice

- **作用**

  投出一个指定面数 （最大1亿面）的骰子。

- **用法**

``` Text
/dice (骰子面数)
```

- **权限/可用范围**

  无。

- **别名**

  `d`。



---

### help

- **作用**

  显示指令的帮助信息。

- **用法**

``` Text
/help <指令名>
```

- **权限/可用范围**

  无。

- **别名**

  `xme帮助`、 `usage`、 `xmehelp`、 `docs`、 `帮助`。

---

### httpcode

- **作用**

  查询 http 状态码并且返回一张来自 [http.cat](https://http.cat) 的猫猫图。

- **用法**

``` Text
/httpcode <状态码>
```

- **权限/可用范围**

  无。

- **别名**

  `http`、 `http状态码`。

---

### prevrecall

- **作用**

  查询 / 启用 / 关闭 群聊的防撤回功能。

- **用法**

``` Text
/prevrecall <开|关|T|F>
```

- **权限/可用范围**

  - **在群聊内**
  - **调用者是 SUPERUSER**

- **别名**

  `防撤回`、 `precall`、 `防撤`、 `防撤回功能`。

---

### status

- **作用**

  查看运行该 XME-Bot 实例的设备的系统状态。

- **用法**

``` Text
/status
```

- **权限/可用范围**

  无。

- **别名**

  `系统状态`、 `stats`。

---

### weather

- **作用**

  查看指定地区的天气。

  天气 API 来源于[高德开放平台](https://lbs.amap.com/api/webservice/guide/api/weatherinfo)。

- **用法**

``` Text
/weather <地区名> <未来天气预测天数(1~3)>
```

- **权限/可用范围**

  无。

- **别名**

  `天气`、 `wea`、 `查看天气`。

---

### wife

- **作用**

  查看你今日的老婆群员。

- **用法**

``` Text
/wife
```

- **权限/可用范围**

  - **在群聊内**

- **别名**

  `今日老婆`。
